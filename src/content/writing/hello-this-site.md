## 它不是什么

不是 AI 教程站，不是 newsletter，不是导航站。

## 它是什么

四个板块，每一块都对应我自己每天会回头查的东西。

- **Library**——我整理过的官方一线资料（Anthropic、Claude 文档、startup 案例）。每一条都附**原文链接 + 我的注解**；不复制原文。完整翻译走 GitHub repo，本站只放摘要与读法。
- **OS**——AI-native Founder OS：四阶段方法论（GUIDE）+ 我反复在用的 [可复制 prompt](/os/prompts)。这套东西是我自己每天工作时会打开的，不是写给人看的「方法论」。
- **Build**——我做过和正在做的项目，硬指标说话。
- **Writing**——这里。短文、读后感、小红书精选的长版本。

## 为什么不用别人现成的笔记/翻译站

因为我每次想用别人整理的版本时，都要重新理解他们的归档逻辑。**我宁可花一周搭一个我自己的归档结构，省下未来一年的查阅时间。**

## 法律姿态

资料库里的所有外部资料：原作者版权不变，本站只做摘要 + 读法注解。完整翻译只放在 [对应 GitHub repo](https://github.com/JasonYeYuhe/ai-native-founder-os-playbook) 并附非官方/收到请求即删的免责声明。请优先支持官方原版。
