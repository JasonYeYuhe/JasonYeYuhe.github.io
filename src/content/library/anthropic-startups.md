## 这是什么

Anthropic 给创业公司开的项目页，包含两部分：

- **Startups Program**：合格创业公司可申请的 API 额度、速率上限、活动邀请等支持
- **Case studies**：跑在 Claude 上的真实创业公司，从 YC 早期项目到行业垂直工具

## 为什么值得读

Playbook 描述的是「应该怎么做」；这里是「别人具体怎么做了」。两者必须互看，否则方法论会停留在抽象。

我推荐的读法：

1. 先在 Playbook 里确认你当前在哪一阶段（Idea / MVP / Launch / Scale）
2. 在 Case studies 里找 2-3 个**同阶段、不同行业**的案例
3. 抽出「他们的工具组合 + 决策节奏」，而不是抄结果

## 我的注

> 看案例最容易踩的坑是「他做成了所以我也照抄」。正确的姿势是把每个案例当作一次反向工程练习：他们在 Idea 阶段做对了什么？在 MVP 阶段克制了什么？
