## 这是什么

Anthropic 给 Claude Code 维护的官方手册——不是教程，是参考资料。每个功能页都是「这个东西是什么 / 怎么配置 / 边界在哪」的快速答案。

## 我每次翻它都是因为

- `CLAUDE.md` 该写些什么、放哪一层（项目级 / 用户级 / 全局）
- hooks / skills / slash commands 的格式细节
- settings.json 里某个不常用键的精确含义
- 接 MCP server 时本地配置怎么写
- 哪些 IDE 集成已经稳定可用

## 我的注

> 官方文档不是用来读完的，是用来「记得在它里面」的。第一次过一遍记结构，之后只在遇到具体问题时回查。把它加入浏览器书签栏比通读一遍价值大十倍。
